# Flappy Bird Game

## Общее описание
Игра Flappy Bird, реализованная на Python с использованием библиотеки Pygame.  
Цель: управлять птицей, пролетая между трубами, и набрать как можно больше очков.

## Файловая структура
- `flappy.py` – главный файл игры, содержит весь код: классы Bird, Pipe, логику игры, отрисовку и управление.
- `Dockerfile` – файл для сборки Docker-образа, позволяет запускать игру в контейнере.

## Дополнительные функции (итеративные улучшения)
1. **Смена темы (день/ночь)**: каждые 10 очков фон меняется с дневного на ночной и обратно.
2. **Система медалей**: после окончания игры выводится медаль (дерево, бронза, серебро, золото) в зависимости от набранных очков.
3. **Пауза**: по клавишам `P` или `ESC` игра ставится на паузу с отображением меню.
4. **Облака**: фоновые облака, движущиеся с разной скоростью.
5. **Рекорд**: запоминается лучший результат за текущую сессию (в памяти).

## Управление
- **Пробел / ЛКМ** – прыжок (или начать игру).
- **P / ESC** – пауза (во время игры).
- **R** – рестарт (в меню паузы).
- **Q / ESC** – выход (в меню паузы).

## Запуск локально
```bash
pip install pygame
python flappy.py
